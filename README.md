# TT Burger Portal de Vendas

## Requisitos

* Docker instalado (https://docs.docker.com/get-docker/)

## Rodando a Aplicação

* Digite o seguinte comando no termina, na pasta raiz do repositório

```
docker-compose up
```

## Url do Portal de Vendas

* http://127.0.0.1:8080/

## Rotas da API

* http://127.0.0.1:5000/
* http://127.0.0.1:5000/vendas
